# Ha7at - Rádiócsillagászat

Üdvözöllek a Ha7at rádiócsillagászati webhelyén! Ez az oldal méréseim, kutatásaim és eredményeim dokumentálására szolgál.

## 🌌 Projekt Leírás

A rádiócsillagászat a csillagok, galaxisok és más kozmikus objektumok rádióhullámok segítségével történő megfigyelésének tudománya. Ez az oldal bemutatja:

- **Mérési Eredmények:** Hidrogénvonal detektálása, napműködés monitorozása, meteor burst
- **Eszközök és Szoftverek:** Antennák, vevőkészülékek, feldolgozó szoftverek
- **Galéria:** Spektrumok, grafikonok, berendezés fotók
- **Technika:** Kiegészítő információk a rádióamatőr technológiáról

## 🎯 Fő Projektok

### 1. Hidrogénvonal (1420 MHz)
A semleges hidrogén által kibocsátott karakterisztikus rádiófrekvenciájú vonal detektálása a galaxiscentrumban.

**Berendezés:** 1,2 m parabolikus tükör, RTL-SDR vevő

### 2. Napműködés Monitorozása (18 MHz)
A Nap rádióemissziójának kontinuus figyelése és az aktivitás nyomon követése.

**Berendezés:** 15 m hosszúdrót antenna, RTL-SDR

### 3. Meteor Burst Detektálása (50 MHz)
Meteorok által visszavert rádióhullám reflexiók detektálása.

**Berendezés:** Yagi antenna 50 MHz-re, USRP B210
